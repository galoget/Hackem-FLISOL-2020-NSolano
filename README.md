# Flisol 2020 Online Hackem - Nelson Solano - Prcesamiento Digital de Señales con Python
## Requerimientos
1. Tener Python almenos con la versión 3.7 instalado en el sistema operativo, en caso de no tenerlo ingresar al siguiente enlace:
https://www.python.org/downloads/
2. Instalar las libreras **jupyter**, **numpy**, **scipy** y **matplotlib**
python -m pip install --user numpy scipy matplotlib ipython jupyter pandas sympy nose

## Uso de la libreta
1. Abrir una terminal y ejecutar **jupyter**:
jupyter notebook
2. Ingresar en el browser y poner la siguiente dirección:
http://localhost:8888
3. Dentro del dashboar ejecutar la libreta PDS.ipynb
